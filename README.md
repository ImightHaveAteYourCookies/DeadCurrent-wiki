# DeadCurrent

**Minecraft Version:** 1.20.1  
**Mod Loader:** Forge  
**Theme:** Industrial survival in an extremely hostile world  

DeadCurrent is a high-difficulty survival modpack where mobs dominate both caves and the night. Traditional mining is dangerous and inefficient. Automation and industrial progression are required to survive and eventually stabilize the world.
