# System Requirements

## Minimum Requirements (Playable)

- **RAM Allocation:** 8 GB
- **Total System RAM:** 16 GB
- **CPU:** Modern 4-core processor (Ryzen 3 / i5 equivalent)
- **GPU:** Dedicated GPU recommended
- **Storage:** SSD strongly recommended

This configuration supports early-to-mid game progression.

---

## Recommended (Intended Experience)

- **RAM Allocation:** 10–12 GB
- **Total System RAM:** 16–32 GB
- **CPU:** 6+ cores (Ryzen 5 / i7 equivalent)
- **GPU:** Mid-range dedicated GPU
- **Storage:** SSD required

Recommended for:
- Late-game automation
- Digital mining
- Large Create factories
- Multiplayer servers

---

## High-End / Server Hosting

For servers with 4–8 players:

- **RAM:** 12–16 GB allocated
- **CPU:** High single-core performance preferred
- **Storage:** NVMe SSD strongly recommended

Automation-heavy bases and large mob events (e.g., Wither Storm) require stable performance headroom.

## Important Notes

- Under-allocating RAM will cause:
  - Chunk loading stutter
  - Factory lag
  - Crash loops during boss events
- Over-allocating RAM (above 14–16 GB) may reduce JVM efficiency.
- Always use Java 17 (required for 1.20.1 Forge).
